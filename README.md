# 📚 JavaScript 기초 제어문

자바스크립트 - 변수 선언, 조건문(if-else), 논리 연산자 활용 실습

---

## 🛠️ 실습 내용

### 1️⃣ 실습 1: 콘솔 계산기 (`prac1.js`)
- **동작**: 두 수와 연산자(+, -, *, /)를 변수로 선언해 계산 결과 출력
- **핵심**: `if ~ else if` 문을 활용한 연산자별 분기 처리

### 2️⃣ 실습 2: 최종 합격 판정 프로그램 (`prac2.js`)
- **동작**: 점수, 출석률, 과제 제출 여부에 따라 합격/불합격 여부 판정
- **합격 기준**:
  - 기본: 점수 ≥ 70 **AND** 출석률 ≥ 80 **AND** 과제 제출 완료
  - 예외: 점수 ≥ 90 이면 출석률 무시 후 합격 (단, 과제 제출은 필수)
- **핵심**: 논리 연산자(`&&`, `||`)의 우선순위 및 복합 조건문 활용

---

## 🚀 실행 방법
`node` 명령어를 통해 터미널에서 결과를 확인 가능
```bash
node prac1.js
```
```bash
node prac2.js
```
